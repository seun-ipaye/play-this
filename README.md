# Play This

**Real-Time song suggestion web-app for parties**

A host creates a party and sends a 'DJ Invite Link' to the DJ. The host gives users a code (code/link/QR code) to join the party room, where they can suggest and upvote songs that they want the DJ to play. Guests get their song suggestions in without bugging the DJ.

**Created by:** Seun Samuel-Ipaye, Kamso Odogwu

## Tech Stack

- **Frontend:** React + Vite
- **Auth & DB:** PostgreSQL
- **Backend:** FastAPI
- **Language:** JavaScript (JSX, no TypeScript), Python, SQL
- **Hosting:** Render, Vercel

## Features

- Real-time suggested song list 
- Interactive DJ Dashboard (can mark songs as played, later, etc)
- Account creation for hosts and DJ's
