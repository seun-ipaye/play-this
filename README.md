# Play This

**Real-Time song suggestion web-app for parties**

A host creates a party and invites a DJ using a DJ invite link.
Guests join using a code, link, or QR code and can suggest and upvote songs in real time.

The DJ dashboard allows the DJ to manage the queue and mark songs as played without being interrupted by guests.

**Created by:** Seun Samuel-Ipaye, Kamso Odogwu

## Tech Stack

- **Frontend:** React + Vite
- **Database:** PostgreSQL
- **Backend:** FastAPI
- **Languages:** JavaScript (JSX), Python, SQL
- **Hosting:** Render, Vercel

## Features

- Real-time song suggestion feed
- Upvoting system to prioritize songs
- DJ dashboard to manage the queue
- Ability to mark songs as played or saved for later
- Host and DJ account support
